Cau 1: Bai tap ve selectors (bo chon)

Yeu cau: Tao 1 thu muc dat ten la css, trong thu muc css tao file styles.css, sau do trong file index.html hay link them file styles.css vao (dang duong dan relative - tuong doi). Tao ra cac doan van duoi day:

Tao the p nhu nay va them class="color-red", sau do viet them css de dong chu nay thanh mau do ( Dung bo chon class - class selector ).

Tao the p nhu nay va them id="color-green", sau do viet them css de dong chu nay thanh mau xanh la ( Dung bo chon id - id selector ).
